# JobTrek 2.0 - Career Sustainability Tool 🚀

> Analyze your job's financial value, work-life balance, and burnout risk with this comprehensive career sustainability calculator.

## ✨ Features

### 1. **Net Income Reality Check** 💰
Calculate your disposable income and financial risk level based on:
- Gross salary
- Tax rate
- BPJS contributions
- Living expenses (rent, food, transport, loans)

**Output:**
- Disposable income amount & percentage
- Financial risk level (Low/Medium/High)

### 2. **Effective Hourly Wage** ⏰
Discover your real hourly wage including all work-related time:
- Monthly salary
- Regular work hours
- Overtime hours
- Commute time

**Output:**
- Real hourly wage
- Market comparison with industry averages

### 3. **Work-Life Balance Score** ⚖️
Assess your work-life balance (0-100 score):
- Weekly work hours
- Overtime frequency
- Sleep duration
- Work days per week

**Output:**
- Score 0-100
- Category: Red/Warning/Stable
- Personalized feedback

### 4. **Burnout Risk Estimator** 🔥
Evaluate your burnout risk level:
- Stress level (1-5)
- Job satisfaction (1-5)
- Leave days used per year

**Output:**
- Risk score (0-100)
- Risk level: Low/Medium/High/Critical
- Actionable recommendations

### 5. **Career Sustainability Index** 📊
Get your final career sustainability score by combining:
- Financial score (35%)
- Time score (35%)
- Burnout score (30%)

**Output:**
- Overall score 0-100
- Category: Excellent/Good/Fair/Poor

### 6. **Anonymous Aggregated Stats** (Coming Soon)
View industry averages and comparisons with:
- Income buckets (not raw amounts)
- Industry type
- Weekly hour ranges
- Sustainability scores

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **State Management**: React Hooks
- **Deployment Ready**: Vercel-compatible

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ (Note: Warnings shown but works fine)
- npm, yarn, or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/jobtrek-id/jobtrek-app.git

# Navigate to project directory
cd jobtrek-app

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📱 Usage

1. **Navigate through the tabs** to complete each calculator
2. **Fill in your information** in each section
3. **View your results** instantly with personalized feedback
4. **Calculate your Career Sustainability Index** after completing all 4 calculators
5. **Get actionable insights** to improve your career sustainability

## 🎯 Use Cases

- **Job Seekers**: Compare job offers beyond just salary
- **Career Changers**: Evaluate career transition decisions
- **Employees**: Assess current job sustainability
- **Freelancers**: Calculate effective hourly rates
- **HR Professionals**: Benchmark job roles and compensation

## 🔒 Privacy

- **All calculations happen locally in your browser**
- **No personal data is stored or transmitted**
- **No account required**
- **Anonymous aggregated stats feature will use bucketed data only**
