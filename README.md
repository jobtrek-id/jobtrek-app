# JobTrek 2.0 🚀

> **Career Sustainability Tool** - Make data-driven decisions about your job and career.

## ✨ Current Feature (MVP)

### 💰 Net Income Reality Check

Calculate your disposable income and understand your true financial health:

- **Input**: Salary, tax, BPJS, living expenses
- **Output**: Disposable income, percentage, risk level
- **Insight**: "Is this job actually sustainable for me?"

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/jobtrek-id/jobtrek-app.git

# Navigate to project
cd jobtrek-app

# Install dependencies
npm install

# Run development server
npm run dev
```

Open http://localhost:3000

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v3
- **UI Components**: shadcn/ui
- **State**: React Hooks

## 🔒 Privacy

✅ **100% Client-Side** - All calculations happen in your browser
✅ **No Data Collection** - We don't track or store anything
✅ **Open Source** - Code is public and auditable

## 📦 Installation

### Prerequisites
- Node.js 18+ (works fine with Node.js 18)
- npm, yarn, or pnpm

### Development

```bash
npm install
npm run dev
```

### Build for Production

```bash
npm run build
npm start
```

## 🌐 Deployment

### Vercel (Recommended)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

The app will be live in seconds.
